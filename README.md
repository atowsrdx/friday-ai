# F.R.I.D.A.Y — AI Voice Assistant

Your personal always-listening AI assistant. Powered by Gemini or Claude.

## 🚀 Deploy to GitHub Pages (5 steps)

1. Create a new GitHub repository (e.g. `friday-ai`)
2. Upload **all files** in this folder to the repo root
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Click **Save** — your app is live at `https://yourusername.github.io/friday-ai`
5. Open it in **Chrome on Android** → tap "Add to Home Screen" for app experience
